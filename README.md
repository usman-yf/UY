## Features

- **Luxury Editorial Design:** Monochromatic color scheme (`#121212`, `#F5F5F5`) with precise typography (Clash Display & Inter) and asymmetric layouts.
- **Cinematic Animations:** Powered by GSAP and ScrollTrigger for text reveals, parallax images, and magnetic UI elements.
- **Smooth Scrolling:** Integrated with Lenis for an ultra-smooth, buttery scrolling experience.
- **Bento Grids & Squircles:** Modern UI patterns avoiding sharp rectangles, utilizing organic shapes and inverted cutouts.
- **Fully Responsive:** Meticulously crafted for seamless experiences across 1920px desktops, tablets, and 320px mobile devices.

## Tech Stack

- **HTML5:** Semantic and accessible structure.
- **Vanilla CSS:** Pure, high-performance styling without heavy frameworks. Custom variables, flexbox, CSS grid, and typography clamping.
- **JavaScript (ES6):** Modular architecture handling interactions and lifecycle state.
- **GSAP (GreenSock):** For timeline animations, scroll-triggered reveals, and micro-interactions.
- **Lenis:** Smooth scroll hijacking tailored for premium web experiences.
- **SplitType:** For advanced character, word, and line-level typography animations.
- **Lucide Icons:** Clean, consistent SVG iconography.

## Structure

## Setup & Deployment

1. **Local Development:** 
   No build step required. Simply open `index.html` in your browser or run a local server (e.g., Live Server extension in VSCode).
2. **Deployment:** 
   Ready to be deployed out-of-the-box to GitHub Pages, Vercel, Netlify, or any static hosting provider.
